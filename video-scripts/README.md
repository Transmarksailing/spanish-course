# Video Scripts — Spanish Course

Per les een drietalig script (ES → EN → NL) van ~2 minuten.
Gebruik deze scripts in **HeyGen**, **Synthesia** of **D-ID** om AI-presentatrice video's te genereren.

## Aanbevolen instellingen
- **Avatar**: Vrouwelijke presentatrice, vriendelijk, professioneel
- **Talen**: Maak 3 versies per les OF 1 video met taalwisselingen
- **Duur**: ~2-3 minuten per les
- **Achtergrond**: Klaslokaal of Mediterraans thema

## Bestanden
Elk `.txt` bestand bevat het volledige script met ES/EN/NL secties.

# Videoscript: Tegenwoordige Tijd — Onregelmatige Werkwoorden

**Les:** Present Tense — Irregular Verbs
**Duur:** ca. 8-10 minuten
**Taal:** Nederlands met Spaanse voorbeelden
**Stijl:** Rustig, duidelijk, herhaling van kernpunten

---

## INTRO (30 sec)

> Hallo en welkom bij Transmark Language Tool!
>
> Vandaag gaan we het hebben over de **onregelmatige werkwoorden** in de tegenwoordige tijd — el **Presente de Indicativo**.
>
> Dit is een van de belangrijkste onderwerpen in het Spaans, want deze werkwoorden gebruik je elke dag.
>
> We behandelen twee groepen:
> 1. **Stamveranderende werkwoorden** — waarbij de klinker in de stam verandert
> 2. **Yo-vorm onregelmatigheden** — werkwoorden die alleen in de "yo"-vorm anders zijn

---

## DEEL 1: STAMVERANDERENDE WERKWOORDEN (4 min)

### Uitleg (1 min)

> Bij stamveranderende werkwoorden verandert de **klinker in de stam** wanneer je het werkwoord vervoegt.
>
> **Belangrijk:** Dit gebeurt in **alle vormen behalve nosotros en vosotros**.
> Nosotros en vosotros houden de normale stam.
>
> Er zijn vier soorten veranderingen:

**[TOON OP SCHERM:]**

```
e → ie       o → ue       e → i       u → ue
```

### Type 1: e → ie (1 min)

> De eerste en meest voorkomende: **e wordt ie**.

**[TOON OP SCHERM: QUERER (willen)]**

```
yo quiero          nosotros queremos
tú quieres         vosotros queréis
él/ella quiere      ellos quieren
```

> Zie je het? **Que-re-mos** — nosotros houdt de gewone **e**.
> Maar **quie-ro**, **quie-res**, **quie-re**, **quie-ren** — overal verandert de e in ie.

> Andere werkwoorden met **e → ie**:

**[TOON OP SCHERM:]**

```
empezar   → yo empiezo      (beginnen)
cerrar    → yo cierro       (sluiten)
pensar    → yo pienso       (denken)
preferir  → yo prefiero     (de voorkeur geven aan)
entender  → yo entiendo     (begrijpen)
perder    → yo pierdo       (verliezen)
```

> **Ezelsbruggetje:** Denk aan een schoen — de "boot" (laars). De vormen die veranderen vormen de vorm van een laars als je ze in een tabel zet. Daarom noemen we ze ook wel **"boot verbs"**.

### Type 2: o → ue (45 sec)

> De tweede groep: **o wordt ue**.

**[TOON OP SCHERM: PODER (kunnen)]**

```
yo puedo           nosotros podemos
tú puedes          vosotros podéis
él/ella puede       ellos pueden
```

> Weer: nosotros **pod-e-mos** — normaal. Maar yo **pue-do** — de o wordt ue.

> Andere werkwoorden:

**[TOON OP SCHERM:]**

```
volver    → yo vuelvo       (terugkomen)
dormir    → yo duermo       (slapen)
costar    → cuesta          (kosten)
almorzar  → yo almuerzo     (lunchen)
recordar  → yo recuerdo     (herinneren)
```

### Type 3: e → i (30 sec)

> De derde groep: **e wordt i**. Dit zijn meestal **-ir werkwoorden**.

**[TOON OP SCHERM: PEDIR (vragen/bestellen)]**

```
yo pido            nosotros pedimos
tú pides           vosotros pedís
él/ella pide        ellos piden
```

> Andere werkwoorden:

**[TOON OP SCHERM:]**

```
servir    → yo sirvo        (serveren/dienen)
repetir   → yo repito       (herhalen)
seguir    → yo sigo         (volgen)
reír      → yo río          (lachen)
```

### Type 4: u → ue (15 sec)

> En tot slot een uniek geval: **u wordt ue**. Dit geldt eigenlijk maar voor één werkwoord:

**[TOON OP SCHERM: JUGAR (spelen)]**

```
yo juego           nosotros jugamos
tú juegas          vosotros jugáis
él/ella juega       ellos juegan
```

### Samenvatting Deel 1 (15 sec)

> Onthoud dus:
> - De stam verandert in **yo, tú, él/ella en ellos** — de "laarsvorm"
> - **Nosotros en vosotros** blijven normaal
> - Vier types: **e→ie, o→ue, e→i, u→ue**

---

## DEEL 2: YO-VORM ONREGELMATIGHEDEN (3 min)

### Uitleg (30 sec)

> Nu de tweede groep: werkwoorden die **alleen in de yo-vorm** onregelmatig zijn.
> De andere vormen zijn gewoon regelmatig!
>
> Er zijn twee belangrijke patronen: de **-go werkwoorden** en de **-zco werkwoorden**.

### De -go werkwoorden (1.5 min)

> Bij deze werkwoorden eindigt de yo-vorm op **-go**.

**[TOON OP SCHERM:]**

```
hacer    → yo hago      (maken/doen)
           tú haces, él hace, nosotros hacemos...

poner    → yo pongo     (zetten/leggen)
salir    → yo salgo     (uitgaan/vertrekken)
traer    → yo traigo    (meebrengen)
valer    → yo valgo     (waard zijn)
```

> En dan zijn er werkwoorden die **twee onregelmatigheden** combineren — de -go vorm PLUS een stamverandering:

**[TOON OP SCHERM:]**

```
decir    → yo digo      (zeggen)     ← -go + e→i
           tú dices, él dice...

venir    → yo vengo     (komen)      ← -go + e→ie
           tú vienes, él viene...

tener    → yo tengo     (hebben)     ← -go + e→ie
           tú tienes, él tiene...

oír      → yo oigo      (horen)
           tú oyes, él oye...
```

> **Tip:** De meest gebruikte zijn: **hago, pongo, salgo, tengo, vengo, digo**. Leer deze zes uit je hoofd!

### De -zco werkwoorden (45 sec)

> Werkwoorden die eindigen op **-cer** of **-cir** krijgen in de yo-vorm **-zco**:

**[TOON OP SCHERM:]**

```
conocer   → yo conozco     (kennen)
producir  → yo produzco    (produceren)
conducir  → yo conduzco    (rijden/besturen)
traducir  → yo traduzco    (vertalen)
ofrecer   → yo ofrezco     (aanbieden)
```

> De rest van de vervoeging is gewoon regelmatig:
> tú conoces, él conoce, nosotros conocemos...

### Overige yo-onregelmatigheden (15 sec)

> En tot slot drie werkwoorden die hun eigen ding doen:

**[TOON OP SCHERM:]**

```
saber  → yo sé       (weten)
dar    → yo doy      (geven)
ver    → yo veo      (zien)
```

## DEEL 3: VOORBEELDEN IN CONTEXT (1.5 min)

> Laten we nu een paar zinnen bekijken om het in de praktijk te zien:

**[TOON OP SCHERM, één voor één:]**

```
1. Yo no puedo hacer nada.
   → Ik kan niets doen.
   (poder: o→ue)

2. ¿Qué piensas tú?
   → Wat denk jij?
   (pensar: e→ie)

3. El bebé duerme en el carricoche.
   → De baby slaapt in de kinderwagen.
   (dormir: o→ue)

4. Yo hago la comida.
   → Ik maak het eten.
   (hacer: yo = hago)

5. Ella prefiere comer pan integral.
   → Zij geeft de voorkeur aan volkoren brood.
   (preferir: e→ie)

6. Yo conduzco el coche.
   → Ik bestuur de auto.
   (conducir: yo = conduzco)
```

## AFSLUITING (30 sec)

> Laten we samenvatten wat we geleerd hebben:
>
> **Stamveranderende werkwoorden:**
> - Veranderen in alle vormen behalve nosotros/vosotros
> - Vier types: e→ie, o→ue, e→i, u→ue
>
> **Yo-vorm onregelmatigheden:**
> - De -go werkwoorden: hago, pongo, salgo, tengo, vengo, digo
> - De -zco werkwoorden: conozco, conduzco, traduzco
> - Speciale gevallen: sé, doy, veo

## TECHNISCHE NOTITIES VOOR OPNAME

**Benodigdheden:**
- Whiteboard of digitaal scherm voor de vervoegingstabellen
- Duidelijke markeerstift/highlighting voor de stamveranderingen (gebruik kleur: oranje voor de verandering)
- Rustige achtergrond, goed licht

**Tips:**
- Spreek de Spaanse woorden langzaam en duidelijk uit
- Benadruk de veranderde klinker door even te pauzeren: "qu-IE-ro" niet "quiero"
- Wijs op het scherm aan waar de verandering zit
- Herhaal het "laars"-patroon visueel — teken eventueel een laars om de veranderende vormen heen

**Software voor opname:**
- **OBS Studio** (gratis) — voor schermopname + webcam
- **Canva** — voor slides met de tabellen (gratis templates)
- Of **PowerPoint/Keynote** — voor eenvoudige presentatieslides
- **DaVinci Resolve** (gratis) — voor montage achteraf
